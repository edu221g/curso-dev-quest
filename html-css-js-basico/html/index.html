HTML - Básico

  • O que são tags em HTML?
    • Tags em HTML são elementos usados para definir e estruturar o conteúdo de uma página da web. As tags são compostas por um nome e podem ter atributos que fornecem informações adicionais sobre o elemento. O conteúdo que está entre as tags é o que aparece na página da web.

      • Exemplos de tags em HTML:

    • <h1>: define um cabeçalho de nível 1. -<img src="imagem.jpg" alt="descrição">: exibe uma imagem na página da web. O atributo `src` especifica o caminho da imagem e o atributo `alt` fornece uma descrição alternativa para a imagem em caso de falha de carregamento.
  • O que são atributos em HTML?

    • Os atributos das tags em HTML fornecem informações adicionais sobre o elemento. Eles são usados para definir propriedades, características e comportamentos dos elementos. Os atributos são adicionados às tags usando a sintaxe nome_do_atributo="valor_do_atributo".

    Exemplos de atributos em HTML:

    • href - usado em uma tag a para especificar o URL do link.
    • <a href="> <https://www.google.com> <a>: cria um link para o Google.
    • class - usado para definir uma classe CSS para um elemento.
    • <p class="destaque">Este texto está em destaque. </p> : define a classe CSS destaque para o parágrafo.
  • O que são meta tags em HTML?

    • Meta tags são elementos HTML que fornecem informações adicionais sobre uma página da web. Elas não são visíveis na página da web, mas são usadas pelos mecanismos de pesquisa e pelos navegadores para entender o conteúdo e o propósito da página. As meta tags são adicionadas à seção head do documento HTML.

    Exemplos de meta tags em HTML:

    • <meta name="description"> <content=>"Uma breve descrição da página.": fornece uma descrição da página que aparece nos resultados da pesquisa.
    • <meta name="keywords" content="palavras-chave, separadas, por, vírgulas">: fornece uma lista de palavras-chave que ajudam os mecanismos de pesquisa a entender o conteúdo da página.
    • <meta name="viewport" content="width=device-width, initial-scale=1.0">: especifica como a página deve ser exibida em dispositivos móveis.
  • O que são títulos e parágrafos em HTML?

    • Os títulos e parágrafos são elementos básicos em HTML para estruturar o conteúdo de uma página da web.

    Títulos

    Os títulos em HTML são usados para definir os cabeçalhos da página. Há seis níveis de títulos, onde <h1> é o maior e <h6> é o menor. A sintaxe é <h1Texto do Título h1>.

    Exemplo:

    • <h1> Título de Nível </h1>
    • <h2>Título de Nível 2 </h2>
    • <h3>Título de Nível 3 </h3>

    Parágrafos

    Os parágrafos em HTML são usados para definir blocos de texto. Eles são criados usando a tag

    . A sintaxe é <p>Texto do Parágrafo </p>.

  • O que são comentários em HTML?

    • Comentários em HTML são trechos de texto que são ignorados pelo navegador e não aparecem na página da web. Eles são usados para adicionar notas e informações para outros desenvolvedores que possam estar trabalhando no código. Os comentários começam com<!-- e terminam com -->.

    Exemplo: <!-- este é um comentário em html --> Os comentários são úteis para documentar o código e explicar a finalidade de determinadas seções de uma página da web. Eles também são usados para desativar temporariamente o código sem removê-lo completamente.

  • O que são Links em HTML e qual sua importância?

    • Em HTML, um link é um elemento que permite que você crie hiperlinks, ou seja, conexões entre diferentes páginas da web, recursos ou seções dentro de uma página. O link mais comumente usado é o elemento <a> (âncora), que é abreviação de "anchor" em inglês.

    A importância dos links em HTML é facilitar a navegação e a interação dos usuários na web. Aqui estão alguns pontos-chave sobre a importância dos links:

    • Navegação entre páginas: Os links permitem que os usuários cliquem em um texto ou elemento específico e sejam levados para outra página da web. Isso permite a criação de uma estrutura de navegação em um site, onde os usuários podem explorar diferentes seções, páginas e recursos com facilidade.
    • Referências a recursos externos: Os links podem ser usados para se referir a recursos externos, como outros sites, imagens, arquivos de áudio, vídeos, documentos, entre outros. Os links permitem que esses recursos sejam acessados diretamente pelos usuários.
    • Compartilhamento de informações: Os links são amplamente utilizados para compartilhar informações na web. Eles permitem que você compartilhe um URL específico com outras pessoas, para que elas possam acessar a mesma página ou recurso que você está visualizando.
    • SEO (Otimização para mecanismos de busca): Os links têm um papel importante na otimização de um site para mecanismos de busca. Os mecanismos de busca analisam a estrutura de links de um site para determinar sua relevância e popularidade. Ter links relevantes e de qualidade pode ajudar a melhorar a classificação de um site nos resultados de pesquisa.
    • Acessibilidade: Os links também são importantes para a acessibilidade na web. Eles permitem que usuários de tecnologias assistivas, como leitores de tela, naveguem facilmente entre diferentes seções de uma página ou acessem recursos externos.

    Em resumo, os links em HTML são essenciais para criar uma experiência de navegação eficiente e interativa na web. Eles permitem que os usuários acessem diferentes páginas e recursos, compartilhem informações e melhorem a usabilidade e a acessibilidade dos sites.

  • Links Externos.

    • Em HTML, um link externo é uma referência a um recurso localizado em um servidor diferente ou em outro local na web. Eles são usados para conectar páginas da web entre si, permitindo que os usuários naveguem para diferentes sites, páginas ou recursos. Os links externos são criados usando o elemento <a> (âncora) e o atributo href (hiperreferência).

    Exemplo

    1. Links para outros sites: Um exemplo simples é criar um link para um site externo. Por exemplo, <a> href="https://www.exemplo.com">Visite o exemplo.com</a> criará um link clicável que levará os usuários ao site exemplo.com quando clicado.
    2. Links para arquivos para download: Você pode criar links para arquivos, como documentos PDF, arquivos de áudio ou vídeo, etc. Por exemplo, <a>href="arquivo.pdf">Baixar arquivo PDF</a> criará um link para um arquivo PDF chamado "arquivo.pdf" que será baixado quando clicado.
    3. Links para anexos de e-mail: Se você estiver criando uma página de contato ou um formulário, pode adicionar um link para permitir que os usuários enviem anexos. Por exemplo, <a> href="mailto:exemplo@exemplo.com">Enviar e-mail com anexo</a> criará um link que abrirá o cliente de e-mail padrão do usuário com um novo e-mail em branco, permitindo que eles anexem arquivos.
    4. Links para folhas de estilo externas: As folhas de estilo externas (CSS) podem ser vinculadas a uma página HTML usando um link externo. Por exemplo, <link rel="stylesheet" href="estilo.css"> vinculará o arquivo "estilo.css" à página HTML, permitindo que você defina a aparência e o layout da página usando CSS.
  • Links Internos.

    • Os links internos em HTML são aqueles que conectam diferentes seções ou partes de uma mesma página da web. Eles são usados para criar navegação dentro de uma página, permitindo que os usuários acessem facilmente diferentes seções de conteúdo. Os links internos são criados usando o mesmo elemento <a> (âncora) e o atributo href, mas o valor do atributo href faz referência a um identificador interno na própria página.

    Aqui estão alguns exemplos práticos de uso de links internos em HTML:

    1. Links para seções da página: Suponha que você tenha uma página longa com várias seções e deseje criar um menu de navegação que permita aos usuários pular para uma seção específica da página. Você pode adicionar links internos para cada seção usando identificadores. Por exemplo: <a> href="#secao1"> Ir para Seção 1</a>

      ... <h2> id="secao1"Seção 1</h2>

      <p>Conteúdo da seção 1</p> Neste exemplo, ao clicar no link "Ir para Seção 1", o usuário será levado diretamente à seção identificada por id="secao1".
    2. Voltar ao topo: Às vezes, é útil ter um link "Voltar ao topo" que permite aos usuários retornarem rapidamente ao início da página. Isso pode ser feito adicionando um link interno com o valor href="#topo" no final da página e um identificador no elemento que representa o topo da página. Por exemplo: ... <a> href="#topo">Voltar ao topo</a> ... <h1> id="topo">Título da página</h1>
